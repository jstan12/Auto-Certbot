# Duo Access Gateway Certbot Certificate Renewal
Renew certificates for Duo Access Gateway generated by Let's Encrypt Certbot. Conf file must be completed before using.
Downloads certificatess to machine that script is ran on. Renewed certificates must be manually uploaded to DAG admin web interface.

Tested on Ubuntu 18.04

### Configuration (conf)

    # SSH user name
    USER="joe"
    # Public URL of DAG
    DAG_URL="dag.example.org"
    # /Users/localUserName/Downloads/ - Must end with "/"
    DOWNLOAD="."
